1. This is a Matlab based interactive program to predict the steady-state positions of main drainage divide (MDD). 
2. The model accepts user-defined input parameters, including uplift rate, erosion coefficient, hillslope width, base-level elevation, and other relevant variables.
3. Ciation：Zhou, C., Tan, X., Liu, Y., Shi, F., 2022. A cross-divide contrast index (C) for assessing controls on the main drainage divide stability of a mountain belt. Geomorphology 398, 108071. https://doi.org/10.1016/j.geomorph.2021.108071.
